+++++++++++++
설명
캡스톤디자인 작품으로 ibm watson을 이용한 논문검색 챗봇이다. 
+++++++++++++
폴더
api - telegram연결, 어플리케이션
node_modules - 코드를 실행시키는데 필요한 모듈
public - ibm 블루믹스에서 public에 노출시키기 위한 코드
util - ibm cloudant db코드
.env - 암호파일
++++++++++++++
팀원별 작성코드
전체팀원 - api 및 연결코드
명지수, 최성욱 - api(파싱)
이소연, 이효진 - api(DB)
대부분의 코딩은 api폴더의 message.js와 telegram폴더의 message.js에서 작성함. 이 외 파일은 ibm watson과 어플리케이션 연결코드
